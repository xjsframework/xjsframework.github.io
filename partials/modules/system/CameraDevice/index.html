CameraDevice class

The CameraDevice Class is the object returned by System Class getCameraDevices method. It provides you with methods to fetch the Camera Device's id, name, and to add it as a source in the current scene.

Basic Usage

var XJS = require('xjs');
var System = XJS.System;

System.getCameraDevices().then(function(cameras) {
  for (var i in cameras) {
    xml = cameras[i].toXML();
    // do something with the XML here
  }
});
