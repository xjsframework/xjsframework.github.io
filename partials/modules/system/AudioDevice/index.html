AudioDevice class

The AudioDevice Class is the object returned by System Class getAudioDevices method. It provides you with methods to fetch the audio device object's attributes, and also provides methods to convert it back to an XML object that is compatible with XBC.

If you are looking to add a microphone device to the stage, please see System/MicrophoneDevice instead.

Basic Usage

var XJS = require('xjs');
var System = XJS.System;

System.getAudioDevices().then(function(audios) {
  for (var i in audios) {
    // Do not include the imaginary xsplit audio device if that ever exist
    if (audios[i].getName().indexOf('xsplit') === -1) {
      xml = audios[i].toXML();
      // do something with the XML here
    }
  }
});
