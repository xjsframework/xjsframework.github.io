Filter class

The Filter class represents a post-processing shader used within XSplit Broadcaster. This can be used to apply effects to videoitems.

Simply use one of the available Filter objects such as Filter.BLUR or Filter.SKETCHPENCILSTROKE as the parameter to the setFilter() method of an Item instance
