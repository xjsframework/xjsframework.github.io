App class

The App Class provides you methods to get and set application-related functionalities.

Basic Usage

var xjs = require('xjs');
var App = new xjs.App();

App.getFrameTime().then(function(frametime) {
  window.frametime = frametime;
});

For methods referring to application audio (i.e. mic and speaker settings, silence detection, etc.). This will affect XBC settings but will not be reflected in the General Settings Window (also will not be persistent after logging out of/exiting the application).
