ImageSource class

The ImageSource class represents the sources of the image items that has been added to the stage. A single source could have multiple items linked into it and any changes to the source would affect all items linked to it.

Each item is represented by the ImageItem class. See: Core/ImageItem

Inherits from: Core/Source

Basic Usage

var xjs = require('xjs');

xjs.Scene.getActiveScene().then(function(scene) {
  scene.getSources().then(function(sources) {
  for (var i in sources) {
      if (sources[i] instanceof XJS.ImageSource) {
        // Manipulate your image source here
        sources[i].setSilenceDetectionEnabled(true);
      }
    }
  })
})
