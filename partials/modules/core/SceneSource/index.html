SceneSource class

The SceneSource class represents the sources of the scene items that has been added to the stage. A single source could have multiple items linked into it and any changes to the source would affect all items linked to it.

Each item is represented by the SceneItem class. See: Core/SceneItem

Inherits from: Core/Source

Basic Usage

var xjs = require('xjs');

xjs.Scene.getActiveScene().then(function(scene) {
  scene.getSources().then(function(sources) {
  for (var i in sources) {
      if (sources[i] instanceof XJS.SceneSource) {
        // Manipulate your scene source here
        sources[i].setSilenceDetectionEnabled(true);
      }
    }
  })
})
